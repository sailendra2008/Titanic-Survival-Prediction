
# Titanic Survival Prediction 🚢

This project predicts whether a passenger survived the Titanic disaster using Logistic Regression.

## Project Overview
- Performed Exploratory Data Analysis (EDA)
- Handled missing values
- Encoded categorical variables
- Trained a Logistic Regression model using Scikit-learn
- Evaluated the model
- Generated predictions on the test dataset
- Submitted predictions to Kaggle

## Results
- Validation Accuracy: 79.89%
- Kaggle Public Score: 0.76555
- Kaggle Leaderboard Rank: 9829

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab
